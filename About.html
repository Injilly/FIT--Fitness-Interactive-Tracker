About FitLife

FitLife is a student-friendly fitness tracker designed to help students maintain a healthy lifestyle while balancing studies.


Features

  • BMI Calculator
  • Workout Tracker with Sets & Reps
  • Workout Streak Counter
  • Exercise Generator
  • Workout Calendar
  • AI Fitness Coach

This project was created as a web development practice project using HTML, CSS and JavaScript.
